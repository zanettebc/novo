Bem-vindo ao Site 1

Tags HTML

A tag <p> é usada para definir parágrafos em HTML. Ela é essencial para estruturar textos e separá-los em blocos.

As tags <h1> a <h6> são usadas para definir cabeçalhos, com <h1> sendo o mais importante e <h6> o menos importante.

A tag <img> é usada para inserir imagens em uma página HTML. Ela requer o atributo 'src' para definir o caminho da imagem.

Estilizações CSS

A propriedade 'color' é usada para definir a cor do texto em CSS. Pode-se usar nomes de cores, códigos hexadecimais, ou valores RGB.

Para centralizar um elemento, pode-se usar 'text-align: center' para texto e 'margin: auto' para elementos de bloco.

A propriedade 'font-size' é usada para definir o tamanho do texto em CSS. Valores podem ser dados em pixels, em, rem, entre outros.
